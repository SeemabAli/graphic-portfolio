# Aown Abbas — Graphic Design Portfolio

A modern one-page portfolio built with **Next.js 15** and **Tailwind CSS 4**.

## Design Direction

**Editorial / Luxury Brutalism** — warm paper tones (`#f5f0e8`) against deep ink black, with a fierce red-orange accent. Bebas Neue display type paired with DM Sans body text. Custom cursor, grain overlay, and ticker marquees add tactile depth.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Sections

1. **Nav** — Fixed glass navbar with smooth scroll links + hire CTA
2. **Ticker** — Animated skills marquee on dark background
3. **Hero** — Large display heading, stats grid, filtered project grid
4. **Projects** — 6 projects with hover reveals and category filtering
5. **About** — Dark section with portrait area + rotating badge
6. **Services** — 6 service cards in a 2-column bordered grid
7. **Testimonials** — 3 client quotes
8. **Contact** — Full contact form with socials
9. **Footer** — Minimal dark footer

## Customization

- Replace `Aown Abbas` with your name globally
- Update projects array in `app/page.tsx`
- Swap placeholder colors/type in project cards with real images using Next.js `<Image />`
- Add your real email to the footer

## Stack

- Next.js 15 (App Router)
- Tailwind CSS 4
- TypeScript
- Google Fonts (Bebas Neue + DM Sans)
- Pure CSS animations — no JS animation library required
